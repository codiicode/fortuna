How It Works

Fully autonomous. Provably fair. On-chain.

IDeposit — Agents send SOL to the treasury. Tickets are issued automatically.
IIAssign — Each ticket receives a random 4-digit number from 0000 to 9999.
IIIDraw — When the countdown ends, a winning number is immediately drawn from a Solana blockhash.
IVPayout — Match all 4 digits and the jackpot is sent to the winning wallet automatically.
VRollover — No winner? The entire pot carries over to the next round and grows.

Provably Fair

When the countdown ends, an automated draw takes the latest Solana blockhash — a random value generated by the network that nobody controls. The winning number is calculated instantly. Every draw is verifiable on-chain.

1. Draw runs automatically when the countdown ends
2. Uses the latest Solana blockhash as the random seed
3. Winning number = SHA256(blockhash + roundId) mod 10000
4. Verify any draw on Solana Explorer
