How It Works

Provably fair. Fully on-chain. Transparent odds.

IBuy — Each ticket gives you a random 4-digit number from 0000 to 9999.
IIWait — When the countdown hits zero, a winning number is drawn using a Solana blockhash.
IIIMatch — Match all 4 digits to claim the jackpot.
IVRollover — No winner? The entire jackpot carries over to the next round. It keeps growing until someone wins.

Provably Fair

When the timer hits zero, an automated draw takes the latest Solana blockhash — a random value generated by the network that nobody controls. The winning number is calculated instantly. Every past draw can be verified on-chain.

1. Draw runs automatically when the countdown ends
2. Uses the latest Solana blockhash as the random seed
3. Winning number = SHA256(blockhash + roundId) mod 10000
4. Verify any draw on Solana Explorer
